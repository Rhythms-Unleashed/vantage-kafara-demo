# VANTAGE — Kafara Village demo

A single-file interactive demo: fly from a globe to a real Sahel village (Kafara, Mali),
then fund holographic structures that materialize from hologram into solid as you give.
All payments are simulated. Open `index.html` (or the GitHub Pages link).

**Imagery:** Sentinel-2 cloudless — https://s2maps.eu by EOX IT Services GmbH
(modified Copernicus Sentinel data), CC BY 4.0. Sky/textures: Poly Haven (CC0). Trees: proctree.js (MIT).
